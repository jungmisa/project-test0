하이퍼 링크

a: anchor(닻의 뜻, 연결을 의미)

href :hyper reference 파일의 경로 주소 등을 의미

a href=링크할 페이지나 파일의 경로 / 하이퍼링크 이름/a

네이버로 이동

  • 절대주소: 경로의 정확한 주소를 표현
    • http://~
    • //경로 또는 /(root) 정확한 파일 위치를 표기
  • 상대주소: html파일과의 상대적인 위치를 표현
    • ./(현재 위치를 표현)
    • ../(현재 위치의 상위 경로를 표현) ../(두 단계 상위 경로를 표현)
  • 이미지에 링크 설정

하이퍼링크 새창에서 열기: target=_blank
