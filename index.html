git

        1.git init
        최초로 한 번 실행
        2.
        폴더에서 최초 한번 스크린샷 생성(git add .)
        스크린샷을 버젼으로 생성
        3. 스크린샷을 버전으로 생성
        git commit -m "커밋메세지"
        4. 원격저장소 생성(폴더별 최초 한번)
        깃허브로 이동하여 new repository 클릭
        5. 로컬저장소와 원격저장소 연결
        pc별 최초 한 번 사용자 등록 이용
        git remote add origin 원격저장소주소(폴더별 1회 실행)
        6. 원격저장소로 코드를 전송
        git push -u origin main(브랜치명)
